### Gulp way


Gulp will run a local webserver, and with it, you can see the project

**Dependencies:**

```bash
node.js with npm
```

Clone the repository.

```bash
git clone git@github.com:matheuslc/desafio-meritt-2.git
```

Install all Gulp dependencies.
```bash
npm install
```

Run gulp
```bash
gulp
```
